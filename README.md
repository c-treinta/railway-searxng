# SearXNG

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/c-treinta/railway-searxng)

Privacy-respecting meta-search engine.

**Services:** `searxng-app`

## Deploy

```bash
make deploy
# A random SEARXNG_SECRET_KEY is generated automatically
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `SEARXNG_SECRET_KEY` | Yes | Random secret (auto-generated by Makefile) |
